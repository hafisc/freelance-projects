# SOW.md — Scope of Work

## Nama Project
MAMITOR 3D — Media Digital Alat Peraga Matriks Minor dan Kofaktor

## Deskripsi Project
Membangun website media digital alat peraga berbasis Next.js dan Three.js untuk membantu pengguna memahami konsep Matriks Minor dan Kofaktor secara interaktif melalui visualisasi 3D.

Project dibuat sebagai website statis interaktif dan dideploy ke Vercel.

## Scope Pengerjaan
1. Membangun ulang website MAMITOR 3D menggunakan Next.js, TypeScript, Tailwind CSS, dan Three.js.
2. Membuat halaman pembuka dengan animasi “Mulai Bermain”.
3. Membuat scene 3D untuk visualisasi matriks 3x3.
4. Membuat input nilai matriks 3x3.
5. Membuat fitur pemilihan baris dan kolom.
6. Membuat animasi visual saat baris dan kolom dipilih.
7. Membuat highlight elemen matriks sisa.
8. Membuat tampilan matriks sisa 2x2.
9. Membuat perhitungan minor matriks.
10. Membuat perhitungan kofaktor matriks.
11. Membuat input jawaban manual untuk pengguna.
12. Membuat fitur cek jawaban.
13. Membuat feedback jawaban benar/salah.
14. Membuat tampilan UI modern, rapi, dan mudah digunakan.
15. Menyesuaikan visual agar terasa seperti alat peraga digital 3D.
16. Membuat tampilan responsive untuk desktop dan mobile.
17. Menggunakan KaTeX untuk menampilkan rumus matematika.
18. Menggunakan Lenis untuk smooth scroll.
19. Menggunakan Framer Motion untuk animasi UI.
20. Melakukan testing fitur utama.
21. Menyiapkan project agar siap deploy ke Vercel.
22. Menyediakan dokumentasi cara install, menjalankan, build, dan deploy.

## Di Luar Scope
- Backend/API.
- Database.
- Login/register.
- Admin panel.
- Payment gateway.
- Multi materi selain Matriks Minor dan Kofaktor.
- Sistem akun siswa/guru.

## Acceptance Criteria
Project dianggap selesai jika:

- Website dapat dibuka tanpa error.
- Tombol “Mulai Bermain” berfungsi.
- Scene 3D tampil normal.
- Pengguna bisa mengisi nilai matriks 3x3.
- Pengguna bisa memilih baris dan kolom.
- Matriks sisa yang ditampilkan sesuai pilihan.
- Minor dihitung dengan benar.
- Kofaktor dihitung dengan benar.
- Cek jawaban berjalan.
- Feedback jawaban tampil jelas.
- Tampilan desktop dan mobile rapi.
- `npm run build` berhasil.
- Website siap deploy ke Vercel.
