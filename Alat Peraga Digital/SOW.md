# SOW.md

## Scope Pengerjaan

1. Melanjutkan website media digital alat peraga dari kode yang sudah diberikan.
2. Merapikan struktur kode HTML, CSS, dan JavaScript agar lebih rapi dan mudah dikembangkan.
3. Memperbaiki bagian kode yang error atau belum berjalan dengan baik.
4. Membuat halaman utama media digital alat peraga MAMITOR 3D.
5. Membuat tampilan interaktif untuk materi matriks minor dan kofaktor.
6. Membuat fitur input nilai matriks 3x3.
7. Membuat fitur pemilihan baris dan kolom.
8. Membuat visualisasi baris dan kolom yang dipilih.
9. Membuat tampilan matriks sisa setelah baris dan kolom dipilih.
10. Membuat perhitungan minor matriks.
11. Membuat perhitungan kofaktor matriks.
12. Membuat fitur input jawaban manual.
13. Membuat fitur cek jawaban.
14. Menambahkan animasi tombol atau halaman “Mulai Bermain”.
15. Memperbagus tampilan website tanpa mengubah konsep utama.
16. Menyesuaikan warna, layout, spacing, dan komponen agar lebih rapi.
17. Membuat tampilan responsive agar nyaman dibuka di laptop maupun HP.
18. Melakukan testing fitur agar perhitungan, tombol, dan tampilan berjalan dengan baik.
19. Deploy website ke Vercel.
20. Menyerahkan source code project.
21. Memberikan dokumentasi singkat cara menjalankan dan deploy project.

## Di Luar Scope

1. Backend database.
2. Login/register user.
3. Dashboard admin.
4. Multi materi selain minor dan kofaktor matriks 3x3.
5. Export PDF.
6. Penyimpanan nilai permanen.
7. Integrasi payment gateway.

## Catatan Pengerjaan

Project dikerjakan sebagai lanjutan dari kode yang diberikan client. Jika kode lama tidak lengkap atau terdapat error, perbaikan dilakukan dengan cara merapikan ulang struktur project tanpa mengubah konsep utama.
