Informasi Sistem Microservice

Sistem ini berjalan dengan arsitektur microservices terdistribusi:

  • Traefik Gateway menerima request di port 80 lalu menyebarkannya sesuai path.
  • REST API digunakan untuk transaksi data (Auth & Trip Booking).
  • WebSockets (Socket.io) digunakan untuk pertukaran pesan chat secara instan serta pembaruan GPS Driver.
  • Monitoring Stack: Anda dapat mengakses visualisasi Grafika Dashboard untuk Host & Container di rute /grafana.
